# Procesos típicos de minería (Flotación, Lixiviación y Electroflotación)

Página web con estética suave/minimalista y colores pastel (fondo beige) que explica procesos típicos de minería de minerales metálicos:
- Flotación
- Lixiviación
- Electroflotación

## Características
- Diseño responsive (desktop y móvil)
- Secciones por proceso con imágenes locales (carpeta `/img`)
- Navegación por anclas (menú superior)
- Estilo minimalista con CSS

## Estructura del proyecto
- `index.html` — página principal
- `styles.css` — estilos
- `img/` — imágenes utilizadas en la página

## Despliegue con GitHub Pages
1. Ir a **Settings → Pages**
2. Seleccionar **Deploy from a branch**
3. Branch: `main` / Folder: `/ (root)`
4. Guardar y esperar el despliegue
